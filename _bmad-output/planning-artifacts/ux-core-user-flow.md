# Apartments-AI MVP: Core Public User Flow

## Overview

A simple, public-facing chat interface for tenants and prospective renters. No account required. Supports tenant-specific branding and multiple buildings.

---

## Step-by-Step User Flow

### 1. Access Tenant-Specific URL

- User lands on a unique URL for their apartment provider/building group.
- Branding (logo, colors, name) is displayed per tenant.

### 2. Select Language

- User chooses preferred language (dropdown or buttons).
- Branding and UI update to selected language.

### 3. Select Apartment/Building (if multiple)

- If tenant has multiple buildings, user selects from a list (cards or dropdown).
- If only one building, skip this step.

### 4. Enter Chat Interface

- User is presented with a clean, intuitive chat UI.
- Branding persists (logo, colors).
- No login/account required.

### 5. Ask Questions

- User types or speaks questions about the apartment/building.
- AI responds with relevant info (amenities, policies, contacts, etc.).
