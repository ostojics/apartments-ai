## Description

Design and implement a sleek, modern chat interface UI for the apartment page. The interface should support receiving and displaying messages in a conversation with an AI agent. Messages are represented as an array of items, each containing message content and a role (either the user or the AI assistant). The UI must be simple, visually appealing, and focused on core chat functionality only. The interface should display these messages properly, clearly distinguishing between user and assistant roles.

## Acceptance Criteria

- Chat interface is implemented on the apartment page
- Supports receiving and displaying messages in a conversation
- Messages are an array of items with content and role (user or AI assistant)
- Interface displays messages properly, distinguishing between user and assistant
- UI is sleek, modern, and visually appealing
- Only essential chat features are present (no extra functionality)
- Code follows product conventions and is mobile responsive
- All new UI/UX is visually verified and tested
